# 需求文档：生产计划与 MRP 增强（60-03）

## 概述

在现有 Production Run 基础上补齐计划层能力：MPS/MRP、能力负载与计划策略。

## 需求

### 需求 1：计划与需求管理
1.1 支持独立需求与版本  
1.2 支持计划策略参数（MTO/MTS）

### 需求 2：MRP 核心能力
2.1 生成计划订单（make/buy）  
2.2 支持批量重算与异常消息  
2.3 支持替代件与损耗率参数

### 需求 3：执行联动
3.1 计划订单可转生产/采购  
3.2 产能负载可视化与预警
