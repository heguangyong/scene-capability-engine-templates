# 需求文档：设备全生命周期增强（60-09）

## 概述

在现有设备与维护模块基础上补齐分类、预防性维护、租赁等能力。

## 需求

### 需求 1：设备分类与状态
1.1 支持设备分层分类  
1.2 支持转移/闲置/报废等状态

### 需求 2：预防性维护
2.1 支持维护策略与计划  
2.2 支持按计划自动生成维护工单

### 需求 3：租赁与履历
3.1 支持设备租入/租出管理  
3.2 支持全生命周期履历追踪
