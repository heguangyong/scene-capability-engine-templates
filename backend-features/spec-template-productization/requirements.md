# 需求文档：Spec 模板产品化（62-06）

## 需求 1：沉淀模板
### 验收标准
1.1 从完成态 Spec 生成模板草案  
1.2 模板覆盖 requirements/design/tasks/docs 全链  
1.3 模板包含能力矩阵与主从协作段落

## 需求 2：模板可用
### 验收标准
2.1 模板可被 `kse templates` 命令识别  
2.2 可用于快速创建新 Spec 并保持治理合规  
2.3 输出模板使用手册
