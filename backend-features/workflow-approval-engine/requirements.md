# 需求文档：工作流与审批引擎（60-10）

## 概述

建立统一审批引擎，支撑订单、采购、变更、费用等跨模块审批。

## 需求

### 需求 1：流程定义
1.1 支持节点、条件、审批人配置  
1.2 支持版本管理与发布

### 需求 2：流程执行
2.1 支持发起、流转、驳回、撤回  
2.2 支持待办与历史查询

### 需求 3：模块接入
3.1 订单/采购/工程变更可接入  
3.2 审批日志可审计
