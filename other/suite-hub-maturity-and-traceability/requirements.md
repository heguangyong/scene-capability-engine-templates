# 需求文档：SuiteHub 成熟度与追溯增强（67-01）

## 概述

本 Spec 聚焦 `SuiteHub` 的治理可见性，补齐“看得见风险、找得到证据”两类能力。

## 需求 1：成熟度与风险趋势卡片

### 验收标准

1.1 每个套件显示成熟度评分  
1.2 每个套件显示风险等级与待补项  
1.3 显示趋势提示以支持下一轮决策

## 需求 2：证据路径追溯入口

### 验收标准

2.1 每个套件展示 Spec 路径列表  
2.2 每个套件展示 Evidence 路径列表  
2.3 支持一键复制路径

## 需求 3：与现有概览共存

### 验收标准

3.1 不破坏能力覆盖率与审计概览  
3.2 异常情况下回退到本地基线  
3.3 样式与当前主题一致
