# 需求文档：Suite 场景画布可视化

## 概述

在 SuiteHub 中新增“场景组合画布”，按三大套件清晰展示模块场景的 ER/BR/DL 与关键流程。

## 需求 1：模块级场景卡片

### 验收标准

1.1 每个模块显示标题与场景描述  
1.2 展示 ER/BR/DL 数量与样例  
1.3 可一键进入模块页面

## 需求 2：场景流程展示

### 验收标准

2.1 显示 keyFlows 关键流程链  
2.2 支持超长流程折叠展示  
2.3 空场景显示默认提示
