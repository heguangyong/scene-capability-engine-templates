# 需求文档：场景编排交互加固（68-02）

## 概述

在场景画布基础上，增强交互效率：过滤、依赖提示、快速路由。

## 需求 1：过滤与聚焦

### 验收标准

1.1 支持按套件过滤场景卡片  
1.2 支持显示“全部”快速还原  
1.3 过滤后交互与路由保持正常

## 需求 2：依赖与跳转提示

### 验收标准

2.1 模块卡显示主要依赖模块  
2.2 提供快速跳转依赖模块入口  
2.3 缺失依赖数据时提供兜底文案
